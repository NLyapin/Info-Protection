Тестовый файл для лабораторной работы №6 - Шифр RSA.

RSA - это криптографический алгоритм с открытым ключом,
основанный на сложности факторизации больших чисел.

Алгоритм:
1. Генерация ключей:
   - Выбираются два простых числа p и q
   - Вычисляется n = p * q
   - Вычисляется φ(n) = (p-1)(q-1)
   - Выбирается e (1 < e < φ(n), НОД(e,φ(n)) = 1)
   - Вычисляется d = e^(-1) mod φ(n)
   - Открытый ключ: (n, e), секретный ключ: (n, d)

2. Шифрование: c = m^e mod n
3. Расшифрование: m = c^d mod n

RSA широко используется в интернете для защиты данных.
