Тестовый файл для лабораторной работы №10 - Электронная подпись ГОСТ Р 34.10-94.

ГОСТ Р 34.10-94 - российский стандарт электронной подписи,
основанный на эллиптических кривых.

Основные особенности:
- Использует эллиптические кривые над полем GF(p)
- Длина подписи: 512 бит (256 бит для r + 256 бит для s)
- Основан на задаче дискретного логарифмирования на эллиптических кривых

Алгоритм подписания:
1. Выбирается эллиптическая кривая E над GF(p)
2. Выбирается точка P на кривой порядка q
3. Выбирается секретный ключ d (1 < d < q)
4. Вычисляется открытый ключ Q = d*P
5. Выбирается случайное число k (1 < k < q)
6. Вычисляется точка R = k*P
7. Вычисляется r = x_R mod q (x_R - координата x точки R)
8. Вычисляется s = (k^(-1)(H(m) + d*r)) mod q
9. Подпись: (r, s)

Алгоритм проверки:
1. Проверяется: 0 < r < q, 0 < s < q
2. Вычисляется v = s^(-1) mod q
3. Вычисляется z1 = (H(m)*v) mod q
4. Вычисляется z2 = (r*v) mod q
5. Вычисляется точка R' = z1*P + z2*Q
6. Вычисляется r' = x_R' mod q
7. Подпись корректна, если r = r'

ГОСТ Р 34.10-94 используется в российских государственных системах.
