Тестовый файл для лабораторной работы №9 - Электронная подпись Эль-Гамаля.

Подпись Эль-Гамаля основана на задаче дискретного логарифмирования
и обеспечивает те же свойства, что и RSA подпись.

Алгоритм подписания:
1. Выбирается простое число p и примитивный элемент g
2. Выбирается секретный ключ x (1 < x < p-1)
3. Вычисляется открытый ключ y = g^x mod p
4. Выбирается случайное число k (1 < k < p-1, НОД(k,p-1) = 1)
5. Вычисляется r = g^k mod p
6. Вычисляется s = k^(-1)(H(m) - x*r) mod (p-1)
7. Подпись: (r, s)

Алгоритм проверки:
1. Проверяется: 1 ≤ r ≤ p-1
2. Вычисляется v1 = g^(H(m)) mod p
3. Вычисляется v2 = (y^r * r^s) mod p
4. Подпись корректна, если v1 = v2

Подпись Эль-Гамаля используется в стандарте DSA.
