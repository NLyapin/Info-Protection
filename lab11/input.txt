Тестовый файл для лабораторной работы №11 - Электронная подпись FIPS 186.

FIPS 186 - американский стандарт цифровой подписи (DSA),
основанный на задаче дискретного логарифмирования.

Основные параметры:
- p: простое число длиной 1024, 2048 или 3072 бита
- q: простой делитель (p-1) длиной 160, 224 или 256 бит
- g: элемент порядка q в группе Z_p*
- x: секретный ключ (1 < x < q)
- y: открытый ключ (y = g^x mod p)

Алгоритм подписания:
1. Выбирается случайное число k (1 < k < q)
2. Вычисляется r = (g^k mod p) mod q
3. Если r = 0, выбирается новое k
4. Вычисляется s = (k^(-1)(H(m) + x*r)) mod q
5. Если s = 0, выбирается новое k
6. Подпись: (r, s)

Алгоритм проверки:
1. Проверяется: 0 < r < q, 0 < s < q
2. Вычисляется w = s^(-1) mod q
3. Вычисляется u1 = (H(m)*w) mod q
4. Вычисляется u2 = (r*w) mod q
5. Вычисляется v = ((g^u1 * y^u2) mod p) mod q
6. Подпись корректна, если v = r

DSA используется в американских государственных системах.
